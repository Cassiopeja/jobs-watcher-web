# Project

This a frontend for backend project [JobsWatcher](https://github.com/Cassiopeja/JobsWatcher).

## Implementation uses

- Vue.js
- Vuex.ORM
- Vuetify
- Vue I18n

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
